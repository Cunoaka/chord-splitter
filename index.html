chordSplitter by HerrmuttLobby

Herrmutt Lobby's Chord Splitter JS is a tool made to split a midi chord coming from any midi device or tracker, sort the notes and send the notes according to their index to different instruments.

Dependencies

Ableton & Max 4 Live (for now)

Install

Works within max4live, refer to the demo ableton live set for usage.
